Swimming World Records throughout History

A race against clocks

This project is an exploration of how much swimming has improved since the beginning of the 20th Century. It includes three different visualizations: the first one focuses in the average speeds of the records per event or modality, the second one analyzes those speeds centering the attention on the swimmers and the last one provides information about range of times and nationality of the swimmers. About this project

This project is part of the master thesis 'Movements and Transformations' about the role that motion plays in visualizations. You can learn more about it here

The data came from the list of world records in swimming set in long course, provided by Wikipedia.
