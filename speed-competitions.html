Swimming World Records throughout History

Swimming records: a competition of speeds

Competition of speeds Speed improvements Analytical visualization

If all records competed against each other in a 50m swimming pool, which one would win?

Visualization of the average speeds of all the records broken from 1900 to the beginning of 2017.

Show information about the speed's calculation

The dot’s speed is a faster representation of the average speeds of all records. It has been calculated by 1 dividing the times to the specific distance of the record’s event (i.e., 50 meters, 100 meters, etc.); and 2 making those speeds relative to the width of the graphics.

About this project

This project is part of the master thesis 'Movements and Transformations' about the role that motion plays in visualizations. You can learn more about it here

The data came from the list of world records in swimming set in long course, provided by Wikipedia.
